# Lang-Chain

Personal scratchpad for learning LangChain and LangGraph.

- `LangChain/` — LangChain basics (chatbots, agents, document loaders)
- `LangGraph/` — agents built on LangGraph (music recommender, weather)

## Setup

```bash
uv sync
```

Add an `OPENAI_API_KEY` to a `.env` file, then run any script, e.g.:

```bash
uv run LangGraph/MusicAgent.py
```
